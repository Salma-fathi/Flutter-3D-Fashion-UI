
# Flutter 3D Fashion UI 👗👟

A professional and innovative Flutter UI project showcasing a fashion application with advanced 3D rotation effects and dynamic color palettes.

## ✨ Features

-   **Enhanced 3D Scroll-Based Rotation**: Experience a smooth 3D rotation of product items (like shoes or handbags) as you scroll through the page, now with added **inertia** for a more natural feel and a subtle **tilt effect** on the X-axis for enhanced realism.
-   **Dynamic Color Palettes with Parallax**: The background gradient seamlessly transitions to match the dominant colors of the displayed product, now featuring a **parallax effect** that adds depth and dynamism to the background as you scroll.
-   **Clean & Modular Code**: Built with best practices, ensuring the code is easy to understand, maintain, and extend.
-   **Network Images**: Utilizes `Image.network` for product images, making the project easier to run and preview without local asset setup.
-   **Responsive Design**: Optimized for various screen sizes, providing a consistent user experience across devices.

## 🚀 Getting Started

### Prerequisites

-   Flutter SDK (>=3.0.0)
-   Dart SDK (>=3.0.0)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Salma-fathi/Flutter-3D-Fashion-UI.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd Flutter-3D-Fashion-UI
    ```
3.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Run the app**:
    ```bash
    flutter run
    ```
